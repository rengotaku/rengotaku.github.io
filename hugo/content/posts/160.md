# 対応
* `path`を付けるのを忘れてませんか？

```:正解
building_path(building)
```

```:間違い
building(building)
```

# 事象
`rails routes` でパスを確認して、viewなどで使用としたら、 `NameError: undefined local variable or method 'path_name'` と怒られる
