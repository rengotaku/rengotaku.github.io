# 事象
```
Autoprefixer doesn’t support Node v4.8.2. Update it
```

# 対応
セットアップ( `https://deb.nodesource.com/setup_10.x` )をしてnodejsをインストールすればよい。
```
curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
sudo apt-get install -y nodejs
```
```!
指定のURLは公式のURL
```
