```sql
INSERT INTO
       table (a, b, c)
   VALUES
       (1, 12, 100)
   ON DUPLICATE KEY UPDATE
        b = 20
        , c = 200;
```
は、重複レコードがなければ `insert into table (a, b, c) values (1, 12, 100)` を行い、重複レコードがあれば `update table set a = 1, b = 20, c = 200` を行う
