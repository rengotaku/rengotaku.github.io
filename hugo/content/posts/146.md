```yaml:ja.yml
ja:
  activemodel:
    models:
      inquiry_form: "お問い合わせフォーム"
      tasks/search: "検索フォーム"
    attributes:
      inquiry_form:
        company_id: 企業ID
      tasks/search:
        created_at_from: 開始日
        created_at_to: 終了日
```
フォームは以下のような感じ
```:app/forms/inquiry_form
class InquiryForm
  include ActiveModel::Model
end
```

```!
moduleの階層を「/」で表す。
activerecordではなく、activemodel。
```

```!
確認は、　@post.errors.full_messages　な感じで。
```
