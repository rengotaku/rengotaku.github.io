```sql:パターン1
CREATE USER IF NOT EXISTS 'user'@'localhost' IDENTIFIED BY 'password';
```

```sql:パターン2
GRANT ALL ON `database`.* TO 'user'@'localhost' IDENTIFIED BY 'password';
```

RDSとかでは下記が一般的？
```sql
GRANT ALL ON `database`.* TO 'user'@'%' IDENTIFIED BY 'password';
```
