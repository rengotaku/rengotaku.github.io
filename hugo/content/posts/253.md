# 対応
```ruby:正
File.open(file_path, 'w+b', 0x644) { |f| f.write file }

===>
--w----r-- 1 300807 14K  1 11 19:03 mail_21667_image_0.pdf
```

```ruby:誤り
File.open(file_path, 'w+b', 644) { |f| f.write file }

===>
--w----r-- 1 300807 14K  1 11 19:03 mail_21667_image_0.pdf
```

# 経緯
初めは、`0644`と記載していたが、lintの指摘で`644`に変更した為。
