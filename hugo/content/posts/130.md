```
# 最初に見つかった <div> を消す
doc.at(:div).&remove
puts doc
puts "-" * 20

# 全ての <br> を消す
doc.search(:br).map &:remove
puts doc
```
```!
atの場合、存在しない場合はnilが返却される
```
