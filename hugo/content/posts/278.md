```
$ grep -lr '<<<<<<<' . | grep -v './.git' | xargs git checkout --ours
or
$ grep -lr '<<<<<<<' . | grep -v './.git' | xargs git checkout --theirs
```

```!
'./.git' を入れているのは、 './.git/objects/pack/pack-xxx.pack' のようなファイルを除外している
```
