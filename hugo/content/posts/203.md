# ymlに記載
```yaml:docker-compose.yml
version: '2'
services:
  web:
    build: ./web
    depends_on:
      - image_api
    ports:
      - 3010:3000
  api:
    build: ./api
    ports:
      - 3020:3000
```

# curlを叩く
```
curl api:3000/path/to/api
```
```!
apiはDocker内で3000ポートでListenしている
```
