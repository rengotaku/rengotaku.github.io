# 解決
[file - How can I tell how many objects I've stored in an S3 bucket? - Stack Overflow](https://stackoverflow.com/a/32908591/12929768) より、下記を実行する
```sh
$ aws s3 ls s3://mybucket/ --recursive | wc -l
```
