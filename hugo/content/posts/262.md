# 事象
いたずらのようなメールが管理しているシステムから飛んでいく。
一週間２回も発生している事に対しクライアントは不服。

# 心象として重要なこと
```
 「現場は今回のことがたまたまだと思っていない。」ということをまずは理解すること。
根本としての報告はこれを払拭しなければならない。

↓たまたと思っていないということは実際どう思っているのか。
・数を稼ぐため、集約で薄い人、いたずらに人が集まるような怪しいことしてんじゃないの？
→調べた結果、どういえるか。

・１週間以内に２度。とあるが、前回の人のメールアドレスはもらった？
・その２度は、同一者？同一IP？　
→　同一者なら多くない（一人だから）と言う。そんな人もいるでしょう。防止難しい。
→　違う人ぽならどこ経由で来た人がいたずら多い傾向？追求できたら、その集客控えます。で鎮火可能。
・今回のアドレスで過去６ヶ月以内に別名（本名含め）で入っていた形跡はない？
・もしくは今回のアドレスで他でもいたずらで使われていたら、このアドレス所有者が恨まれている？

---
フラッシュだから浅はかにしか掘れてないけど、想像してネタをそろえてコミュニケーションとって。想像しておかないとカウンタートーク何も出ない状況に陥る。とにかく真摯に対応し安達さんはグリップはずれないよう。
```
