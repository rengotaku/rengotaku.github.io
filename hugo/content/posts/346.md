```rb
module Foo
  BAR = :original
end

Foo.stub_const(:BAR, :stubbed) do
  Foo::BAR
end
# => :stubbed

Foo::BAR
# => :original
```

> [adammck/minitest-stub-const: Stub constants for the duration of a block in MiniTest](https://github.com/adammck/minitest-stub-const)
