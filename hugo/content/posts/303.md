# 対応
DNSに下記を設定する
```
タイプ: A
名称: @
TTL: 3600
値: サーバーIP
```

## トラブルシューティング
FQDNの値を確認したい場合
```:/etc/kusanagi.d/profile.conf
[WordPress]
PROFILE="WordPress"
KUSANAGI_TYPE="WordPress"
KUSANAGI_FQDN="xxx"
KUSANAGI_DIR="/home/kusanagi/WordPress"
KUSANAGI_DBNAME="xxx"
KUSANAGI_DBUSER="xxx"
KUSANAGI_DBPASS="xxx"
WPLANG="ja"
OPT_WOO=""
KUSANAGI_MARIADB="yes"
KUSANAGI_PSQL="no"
```
