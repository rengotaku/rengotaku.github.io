ansibleで"yum lockfile is held by another process"が発生

July 17, 2019

事象#

yum lockfile is held by another process でfailedする。

解決#

lock_timeout を追加

- name: Install stuff
  yum:
    name:
    - git
    - python36
    lock_timeout: 180

[Ansible] “yum lockfile is held by another process “エラーの対処法｜スクショはつらいよ

バージョン2.8から発生する。
