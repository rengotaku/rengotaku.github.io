rubocop_todo.ymlのルールが適用されない

June 14, 2019

解決#

してない。 複数のルールをマージする機能がないと実現しないが、その機能は外されたらしい。

バージョン#

$ rubocop -v
0.64.0

事象#

inherit_from: .rubocop_1.yml

AllCops:
  Exclude:
    - b.rb

AllCops:
  Exclude:
    - a.rb

上記の場合に、Excludeが適用されるのは、.rubocop.ymlとなる。 両方のファイルを除外して欲しい。

補足#

inherit_modeを使用する#

機能しない

inherit_settings:
  inherit_mode:
    override:
      - 'Enabled'
  duplicate_warnings: false

Lint/HandleExceptions:
  inherit_mode:
    override:
      - Enabled

inherit_mode Documentation Improvements · Issue #5640 · rubocop-hq/rubocop
