::ClassNameの意味

June 26, 2019

定数のスコープ演算子で、トップレベルの定数であることを示す。

A = "::A"

class Foo
    A = "Foo::A"

    class Bar
        A = "Foo::Bar::A"

        p A         # => "Foo::Bar::A"
        p Foo::A    # => "Foo::A"
        p ::A       # => "::A"
    end
end
