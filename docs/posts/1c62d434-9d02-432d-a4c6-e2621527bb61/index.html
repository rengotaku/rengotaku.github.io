submoduleをcloneする

January 21, 2020

clone時#

recursive を付与

$ git clone --recursive git://github.com/foo/bar.git

既にcloneしている#

$ git submodule update --init --recursive

gitでクローンと同時にサブモジュールを初期化、アップデートする – IsaB
