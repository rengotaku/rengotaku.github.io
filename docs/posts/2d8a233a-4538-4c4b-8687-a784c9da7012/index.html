git push を引数なしで実行したい

November 21, 2018

pushする際に -u をつける#

$ git push -u origin master

set-upstreamを確認#

$ git branch -vv
  development                                              4dc13cd3 [origin/development: behind 72] Merge pull request #0000 from xxxx/feature/xxxx

git pull を –set-upstream-to で引数無しで実行可能にする - Qiita
