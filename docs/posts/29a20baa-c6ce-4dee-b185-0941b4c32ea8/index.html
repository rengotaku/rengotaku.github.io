October 31, 2018
動作 キー
移動履歴を進む Command + ]
移動履歴を戻る Command + ]
未読一覧 Command + Shift + A
スレッド一覧 Command + Shift + T
右サイドバーの開閉 Command + .
全メッセージを既読にする Shift + Esc
次の未読チャンネルまたはDM Option + Shift + ↑(or ↓)
スニペットの作成 Cmd + Shift + Enter
ワークスペースのスイッチ Command + [ (or ]) 
"Cmd + /" でショートカット一覧を表示できます。
