履歴を追いやすいリッチクライアント「tig」

February 5, 2019

インストール#

mac#

$ brew install tig

CentOS#

$ yum install tig

コマンド#

履歴をリッチなビューで確認する#

$ tig

ファイルの各行が誰の何のコミットか分かる#

$ tig blame file_name

gitクライアント「tig」を使いこなすための最低限覚えるべき6つのコマンド | 株式会社リッチメディア
